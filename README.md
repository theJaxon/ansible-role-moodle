Ansible Role: Moodle
=========

Install moodle LMS V 3.9.2 on Ubuntu Bionic 18.04

---

Moodle requires 
- PHP 7.2
- MariaDB 10.2.29 or MySQL 5.6 or Postgres 9.5 or MSSQL 2012 or Oracle 11.2

Example Playbook
----------------
```yml
- hosts: all
  roles:
      - moodle
```
